## ¿Cómo puedo acceder?
En RepoSocial, como todas sus funcionalidades giran en torno a GitHub, es imprescindible iniciar sesión con una cuenta de esta plataforma, no hay un login clásico.
Al principio, me parecía lo más lógico, pero después de haber quitado el login corriente hace tiempo he caido en que haría falta una cuenta de Administrador de prueba, pero no es posible, habría que iniciar sesión con GitHub y cambiar el rol desde la Base de Datos.
